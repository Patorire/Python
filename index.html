Python

¿Qué es Python?

Python es un lenguaje de programación interpretado y multiplataforma cuya sintaxis favorece un código legible. Es un lenguaje de programación multiparadigma ya que este soporta orientación a objetos, programación imperativa y también programación funcional. Al ser un lenguaje de propósito general podemos encontrar aplicaciones prácticamente en todos los campos científico-tecnológicos:

  • Análisis de datos.
  • Aplicaciones de escritorio.
  • Bases de datos relacionales / NoSQL.
  • Buenas prácticas de programación / Patrones de diseño.
  • Concurrencia.
  • Criptomonedas / Blockchain.
  • Desarrollo de aplicaciones multimedia.
  • Desarrollo de juegos.
  • Desarrollo de dispositivos embebidos.
  • Desarrollo web.
  • DevOps / Administración de Sistemas / Scripts de automatización.
  • Gráficos por ordenador.
  • Inteligencia artificial.
  • Internet de las cosas.
  • Machine Learning.
  • Programación de parsers / scrapers / crawlers.
  • Programación de redes.
  • Propósitos educativos.
  • Prototipado de software.
  • Seguridad.
  • Tests automatizados.

Historia de Python

Python es un lenguaje de programación de alto nivel creado a finales de los 80/ principios de los 90 por Guido Van Rossum, holandés que trabajaba por aquella época en el Centro para las Matemáticas y la Informática de los Países Bajos. Sus instrucciones son muy cercanas al lenguaje natural en inglés y se hace hincapié en la legibilidad del código. Python fue creado como sucesor del lenguaje ABC.

Ventajas

El ecosistema de Python

En Python existen decenas de paquetes que conforman su ecosistema, cada uno orientado a ámbitos muy específicos, como Machine Learning, minería de datos, web scrapping, tratamiento de imagenes, interacción con harware, generación de API REST o administración y monitorización de servidores que pueden ser usados conjuntamente con nuestro framework Python.
Existen varios frameworks entre los más conocidos está Django o Flask que permiten realizar aplicaciones web realmente complejas, escalables, mantenibles y seguras.

Variantes

Hay ciertas implementaciones de Python según el lenguaje de programación que se ha usado para desarrollarlo.

  • CPython: CPython es la implementación por defeto y más utilizada del lenguajde Python. CPyhton interpreta y ejecuta código Python y proporciona funcionalidad y características principales de Python. Además al ser tan utilizado tiene acceso al ecosistema más aplio de bibliotecas y paquetes, incluidas bibliotecas como NumPy, Django, Flask, TensorFlow...
    CPython tiene un entorno de ejcución con gestión dinámica de memoria. CPython ofrece tiene la capacidad de gestionar errores y excepciones a través de bloques try-except.
    CPython soporta programación asíncrona pero no programación paralela.
  • MicroPython:MicroPython está diseñado para un uso general (ordenadores de sobremesa, servidores, sistemas embebidos...). MicroPython no requiere un sistema operativo, puede ser ejecutado de manera eficaz en un microcontrolador. Aunque MicroPython esté escrito en C, el objetivo que tiene MicroPython es proporcionar un tiempo de ejecución rápido para dispositivos menos potentes, por lo que utiliza menos memoria y sólo proporciona módulos y bibliotecas específicos para sistemas embebidos.
  • Circuito Python: Circuito Python es una bifurcación de MicroPython creado especialmente para ser más sencillo para principiantes. Realmente las diferencias entre Circuito Python y MicroPyhton son mínimas.(Página de documentación).
  • Jython: Jython proporciona las características de productividad de un lenguaje de script maduro, y a diferencia de Python, este se ejecuta en cualquier entorno que soporte una máquina virtual Java (JVM). Las bibliotecas Java de la máquina virtual Java están disponibles para utilizarlas cuando se escriben programas. Con Jython te puedes beneficiar de esa diferencia y utilizar la sintaxis y la mayoría de las características del lenguaje de Python.

    Algunas Curiosidades

    1. ¿Por qué se llama Python?: Según su creador Guido Van Rossum el nombre del lenguaje proviene de una serie de comedia británica llamada "Monty Python's Flying Circus". La comedia se emitió en la BBC durante la década de 1970 y le dio al creador alguna forma de entretenimiento durante el desarrollo de lenguaje. Además, Van Rossum quería un nombre corto y misterioso, siendo Python el ideal para él.
    2. Python se enseña más que el francés: En 2015 Python superó al francés como los lenguajes más populares que se enseñan en las escuelas primarias. Según estadísticas 6 de cada 10 padres preferian que sus hijos aprendieran Python en lugar de francés. Un 75% de los niños prefieren aprender a controlar un robot que aprender francés.
    3. Apps famosas: Muchas de las apps famosas de hoy en día estan desarrolladas con Python o al menos gran parte del código. Algunas de estas aplicaciones son Instagram, Pinterest, Dropbox, BitTorrent, Google App Engine, Ubuntu Software Center...
    4. El logo: El creador de el logo es Tim Parkin y se basó en 2 pitones, una de color azul y otra de color amarillo, basado en la representación de las serpientes mayas.
